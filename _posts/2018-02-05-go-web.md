##### HelloWorld

```go
package main
import (
  "net/http"
  "fmt"
)

func main() {
  http.HandleFunc("/", func(writer http.ResponseWriter, request *http.Request) {
    fmt.Fprintf(writer, "Hello World.\n")
  })
  http.ListenAndServe(":9090", nil)
}
```

> 运行 & localhost:9090 浏览器将输出 `Hello World` 可以看到Go写一个Web Server非常简单
